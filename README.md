## 🔧 Installation
```bash
# Clone the repository
git clone https://github.com/yourusername/AarusWebsite.git

# Navigate to the project directory
cd AarusWebsite

# Install dependencies
npm install
```

## 🖥️ Usage
```bash
# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
